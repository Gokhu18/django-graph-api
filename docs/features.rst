Django Graph API features
=========================

This is a rough guide and not an exhaustive list. Please update this list as you work on adding new features.

Supported
---------

Operations
^^^^^^^^^^
- queries

Types
^^^^^^^^
- objects
- related field
- many related field
- bool, int, float, str, id scalar fields

Unsupported
-----------

Operations
^^^^^^^^^^
- mutations
- subscriptions

Types
^^^^^
- list of scalars field
- interfaces
- unions
- enums
- non-null
- inputs

Querying
^^^^^^^^
- variables
- filtering
- introspection
- aliases
- fragments
- pagination
